# AGENTS.md

Esta pasta contém documentação completa e detalhada. Ela deve ser lida sob demanda, quando as informações em `../design-acesso-rapido/` forem insuficientes.

- Não leia todos os documentos desta pasta por padrão.
- Consulte apenas o arquivo necessário quando houver dúvida específica, ambiguidade, conflito de decisão crítico ou necessidade de detalhe de domínio.
- Use esta pasta para resolver conflitos entre implementação, backlog e sínteses de `../design-acesso-rapido/`.
- Consulte esta pasta quando a tarefa alterar regras de negócio, permissões, estoque, requisições, importação SCPI ou critérios de aceite.
- Em caso de conflito com a síntese rápida, esta documentação completa prevalece, salvo decisão posterior registrada.
- Ao alterar regra de negócio, atualize também a síntese correspondente em `../design-acesso-rapido/`, quando aplicável.
